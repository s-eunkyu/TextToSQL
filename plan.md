\## Day 1 Summary



\- Docker Desktop + WSL2 환경을 구성하고 PostgreSQL 컨테이너를 실행했다  

\- FastAPI 서버를 구축하고 SQLAlchemy로 Postgres 연결을 완료했다  

\- customers, orders 샘플 테이블을 생성하고 실제 데이터를 삽입했다  

\- /health, /db-check, /sample API를 통해 DB 조회가 정상 동작하는 것을 확인했다  

\- Python 가상환경 구성 및 패키지 의존성을 정리했다  

\- Git 초기화 후 GitHub 원격 저장소에 첫 커밋을 완료했다  



결과적으로 로컬 환경에서 API → DB까지 이어지는 기본 백엔드 파이프라인을 완성했다.
